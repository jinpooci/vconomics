# Vconomics Reff Bot
Vconomics Account Creator

## Features
- Automatically create shell account
- Auto get random email domain from generator email
- Error handling
   
## Installation
Requires PHP v7+

```
$ git clone https://github.com/verssache/vconomics
$ cd vconomics
$ php run.php
```
